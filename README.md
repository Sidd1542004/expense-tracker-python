#Virtusa Mini Projects
Welcome to the **Virtusa Mini Projects Repository**
This repository contains three mini-projects developed as part of hands-on learning in **Database Systems, Java Development, and Python Programming**.
---
##  Projects Included
### 1. Java Banking System (GUI + MySQL)
A full-fledged banking system simulation built using **Core Java, Swing GUI, and MySQL**.
#### Key Features
* User Registration & Secure Login
* Deposit, Withdraw & Fund Transfer
* Transaction History (JTable UI)
* Login History Tracking
* MySQL Database Integration (JDBC)
#### Tech Stack
* Java (Core + OOP)
* Swing (GUI)
* JDBC
* MySQL
-------------------------------

### 2.Expense Tracker (Python)
A smart expense tracking application with both **CLI and GUI support**, including analytics and budgeting.
#### Key Features
* Add & manage daily expenses
* Monthly and category-wise analysis
* Budget tracking with alerts
* Insights & spending suggestions
* GUI using tkinter
#### Tech Stack
* Python
* pandas, matplotlib
* tkinter
* CSV file handling
-------------------------------

### 3.Digital Library Audit System (SQL)
A database-driven system designed to manage and analyze **library operations**.
#### Key Features
* Book issue & return tracking
* Overdue detection & penalty calculation
* Book popularity analysis
* Inactive student identification
* SQL Views for reporting

#### Tech Stack
* MySQL
* SQL (DDL, DML, Joins, Views)
-------------------------------
